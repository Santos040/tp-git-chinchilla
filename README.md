\# Proyecto de Sistema de Gestión de pedidos

el sistema de gestión de pedidos para el resto bar “Black Sheep” se ha definido con un alcance claro y delimitado, centrado en la organización de consumos por mesa y el control de stock, dejando fuera de consideración la facturación y otros procesos administrativos avanzados.
esto se permitira atravez del uso de f=diferentes herramientas tanto software como harware, incluyendo una base de datos, una pagina en la cual se visualizara la carta del establecimiento.
